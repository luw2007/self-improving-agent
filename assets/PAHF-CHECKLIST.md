# PAHF 执行检查单

## 每次任务开始

- [ ] 是否先检索该用户的偏好记忆（若存在）？
- [ ] 任务是否涉及主观偏好/个性化选择？
- [ ] 若有歧义：是否只问一个关键澄清问题？
- [ ] 澄清答案是否在执行前写入偏好记忆？

## 执行时

- [ ] 决策是否同时参考：指令、环境事实、偏好摘要、澄清问答？
- [ ] 若偏好与硬约束冲突：是否明确说明冲突点与备选？

## 任务结束

- [ ] 用户是否给出“不满意/不对/偏好变了”等纠错反馈？
- [ ] 若有：是否先做“显著性检测”（过滤无信息反馈）？
- [ ] 若有显著信息：是否将其提炼为单条可复用偏好 note？
- [ ] 新 note 是“新增”还是“覆盖/合并”，是否合理？
